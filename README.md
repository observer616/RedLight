# RedLight
Graduation Project

毕业设计 - 红光社区住宅销售系统

技术：
1. 前端：Thymeleaf+BootStrap+Echarts
2. 后端：Spring Boot
3. 数据库：MySQL

功能模块：用户模块、管理员模块、住宅模块、看房模块、留言模块
其他：权限管理（HandlerIntercepterAdapter）、分页（PageHelper）
